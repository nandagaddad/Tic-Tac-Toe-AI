# Tic-Tac-Toe AI (Python)

This project implements a **Tic-Tac-Toe game in Python** where a human player competes against an AI.  
The AI uses the **Minimax algorithm** to determine the best possible move and play optimally.

---

## Features

- Human vs AI gameplay
- AI makes optimal decisions using the **Minimax algorithm**
- Detects win, loss, or draw conditions
- Simple command-line interface

---

## Technologies Used

- Python
- Minimax algorithm
- Standard Python libraries

---

## How the Game Works

- The game board is a **3 × 3 grid**.
- The **human player uses 'O'**.
- The **AI uses 'X'**.
- Players take turns placing their symbol on the board.
- The AI evaluates possible moves using the **Minimax algorithm** to choose the best move.

---

## How to Run

1. Make sure Python is installed.
2. Run the program using:

```
python tic_tac_toe.py
```

3. Enter your move as **row and column numbers**.

Example:

```
Enter your move (row and column): 1 2
```

---

## Game Outcome

The program checks for:

- AI win
- Player win
- Draw

and prints the result at the end of the game.
